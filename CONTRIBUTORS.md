# Contributors

This repository follows the **Pelorus Marine** open-source mission: **transparent**, **community-based**, **not commercialized**, **Rust-first** runtime software with **maximum practical safety**.

Authoritative policy and credits template: **[platform `CONTRIBUTORS.md`](https://github.com/pelorus-marine/platform/blob/main/CONTRIBUTORS.md)** on GitHub.

Credits for this tree: **`git shortlog -sn`** (and merge commits / reviews). Crate **`authors`** metadata uses the shared Pelorus Marine contributors line — see each **`Cargo.toml`**.
