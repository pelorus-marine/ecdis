# s-126

**IHO S-126** — Marine physical environment

S-100 product for **marine physical environment** (distinct from S-104 water-level focus in current IHO naming—verify edition).

This repository targets **[Pelorus](https://sevenseas.io/pelorus) ECDIS / S-100** tooling. The **s-126** crate is an intentionally minimal **0.0.1** release to reserve the name on [crates.io](https://crates.io/crates/s-126) and align with hyphenated **S-xxx** naming. Semantic decoding may land in later versions.

Normative text always comes from **IHO publications** for the edition you certify against.
