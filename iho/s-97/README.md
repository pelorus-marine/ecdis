# s-97

**IHO S-97** — Guidelines for S-100 product specifications

IHO guidance for authors of S-100-based product specifications; this crate is a placeholder for future tooling or type stubs aligned with [S-97](https://iho.int/).

This repository targets **[Pelorus](https://sevenseas.io/pelorus) ECDIS / S-100** tooling. The **s-97** crate is an intentionally minimal **0.0.1** release to reserve the name on [crates.io](https://crates.io/crates/s-97) and align with hyphenated **S-xxx** naming. Semantic decoding may land in later versions.

Normative text always comes from **IHO publications** for the edition you certify against.
