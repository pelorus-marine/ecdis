# s-98

**IHO S-98** — Data product interoperability

Companion to **S-100** for **how multiple S-100 products interoperate on ECDIS and navigation systems** (display coordination, clutter, conflicts)—not a chart cell format. See [IHO S-98](https://iho.int/).

This repository targets **[Pelorus](https://sevenseas.io/pelorus) ECDIS / S-100** tooling. The **s-98** crate is an intentionally minimal **0.0.1** release to reserve the name on [crates.io](https://crates.io/crates/s-98) and align with hyphenated **S-xxx** naming. Semantic decoding may land in later versions.

Normative text always comes from **IHO publications** for the edition you certify against.
