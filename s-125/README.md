# s-125

**IHO S-125** — Marine aids to navigation (AtoN)

S-100 product for **AtoN** information.

This repository targets **[Pelorus](https://sevenseas.io/pelorus) ECDIS / S-100** tooling. The **s-125** crate is an intentionally minimal **0.0.1** release to reserve the name on [crates.io](https://crates.io/crates/s-125) and align with hyphenated **S-xxx** naming. Semantic decoding may land in later versions.

Normative text always comes from **IHO publications** for the edition you certify against.
